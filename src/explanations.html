Bienvenue dans la section d'aide du jeu !

Datagramme est un jeu de plateau dans lequel vous devez répondre à des questions pour vous frayer un chemin et réparer un réseau endommagé.

Les deux modes de jeu

Il existe deux modes de jeu, dans le premier, le joueur ou l'équipe ayant collecté le plus de Datagrammes - acquis en répondant correctement aux questions - au bout d'un certain nombre de tours remporte la partie.
Dans le deuxième mode de jeu, c'est le premier joueur ou la première équipe arrivé(e) à la sortie, en bas à gauche du plateau qui l'emporte.

Les trois types de cases

Il existe trois types de cases dans Datagramme.
Tout d'abord, les cases rouges représentent les cases question. Si vous répondez correctement à une question, la case devient verte, et vous gagnez un Datagramme. Lorsque la case est verte, la section est réparée et ne donne plus de question.
Ensuite, les cases noires sont des cases vous faisant tirer des cartes Bonus/Malus. Leur effet est imprévisible.
Enfin, les dernières cases sont les routeurs étoilés, aussi appelés cases MEMORY. Lorsque vous tombez sur une telle case, son symbole caché se retourne, vous devez ensuite le retrouver parmi les autres routeurs. Une fois que vous avez trouvé deux routeurs avec le même symbole, ces derniers se connectent, et vous pouvez les traverser pour vous retrouver dans une autre zone du réseau.

Déroulement d'un tour

Chaque joueur ou équipe va jouer à tour de rôle. Tout d'abord, il faut lancer les dés binaires. Le chiffre résultant indique le nombre de cases maximum que vous pourrez parcourir. Suite à cela, les cases accessibles seront alors en surbrillance, il vous suffira de cliquer sur l'une d'elles pour vous y rendre. Ensuite, une pop-up vous indiquera la marche à suivre. Si vous tombez sur une case rouge, une question vous sera posée, il vous suffira d'y répondre pour finir votre tour. Si vous tombez sur une case noire, il vous faudra alors cliquer sur le tas de cartes Bonus/Malus pour en piocher une. Enfin, si vous tombez sur une case MEMORY, il vous faudra cliquer sur un autre routeur pour retourner son symbole. Si les deux symboles concordent, vous venez de créer une nouvelle route. Sinon, les deux symboles sont de nouveau cachés. Votre tour prend fin quoiqu'il advienne.
